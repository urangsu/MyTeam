# 🛠️ MyTeam AI Skills Guide

이 파일은 현재 프로젝트에 설치된 모든 AI 스킬들을 한눈에 파악하기 위해 생성되었습니다. 각 상황에 맞춰 적절한 스킬을 호출해 보세요.

---

### 🚀 배포 및 자동화 (Ship & Automation)
| 스킬명 | 설명 | 추천 상황 |
| :--- | :--- | :--- |
| **`ship`** | 테스트, 버전업, CHANGELOG 업데이트, PR 생성까지 한 번에 진행 | "배포 준비됐어", "PR 올려줘" |
| **`autoplan`** | CEO, 디자인, 엔지니어 리뷰를 자동으로 수행하고 의사결정 | "자동으로 계획 검토해줘" |
| **`land-and-deploy`** | PR을 병합하고 프로덕션 배포 후 상태 확인 | "배포 완료해줘" |

### 🔍 디버깅 및 분석 (Debug & Analysis)
| 스킬명 | 설명 | 추천 상황 |
| :--- | :--- | :--- |
| **`investigate`** | 버그의 근본 원인을 체계적으로 조사하고 수정안 제안 | "이거 왜 안돼?", "디버깅 시작해" |
| **`codex`** | 강력한 에이전틱 코딩 모델에게 제2의 의견을 물음 | "코덱스한테 물어봐줘" |
| **`graphifyy`** | 프로젝트 전체를 지식 그래프로 만들어 복잡한 관계 분석 | "코드 구조 분석해줘" |
| **`health`** | 테스트, 린트 등 코드 품질 점점 및 점수 산출 | "현재 코드 상태 어때?" |

### 🎨 디자인 및 UI (Design & UX)
| 스킬명 | 설명 | 추천 상황 |
| :--- | :--- | :--- |
| **`design-consultation`** | 브랜드 가이드라인 및 디자인 시스템(DESIGN.md) 구축 | "디자인 컨셉 잡아줘" |
| **`design-shotgun`** | 여러 개의 디자인 시안을 한꺼번에 생성해 비교 | "시안 좀 보여줘" |
| **`design-html`** | 승인된 디자인을 실제 HTML/CSS 코드로 구현 | "디자인 코드로 만들어줘" |
| **`design-review`** | 디자인 일관성, 간격, 계층 구조 시각적 감사 | "디자인 검수해줘" |

### 🛠️ 프로젝트 관리 (Management & Utilities)
| 스킬명 | 설명 | 추천 상황 |
| :--- | :--- | :--- |
| **`checkpoint`** | 현재 작업 상태를 저장하고 나중에 다시 불러옴 | "작업 저장해둬", "어디까지 했지?" |

| **`learn`** | 그동안 배운 프로젝트 지식 검색 및 관리 | "예전에 어떻게 했었지?" |
| **`retro`** | 한 주간의 작업 내용 분석 및 회고 보고서 생성 | "이번 주 회고해줘" |

### 💡 팁: 어떻게 "알아서" 쓰게 하나요?

저(Antigravity)나 코덱스는 **상황에 맞는 단어**를 들으면 알아서 해당 스킬을 제안하거나 실행합니다.

*   "배포해줘" → **`ship`** 제안
*   "에러나는데 봐줘" → **`investigate`** 실행
*   "UI 깔끔하게 다듬어줘" → **`design-review`** 실행

따라서 스킬 이름을 외우기보다, **"하고 싶은 일"을 자연스럽게 말씀하시면** 제가 가장 어울리는 스킬을 골라 사용하겠습니다!
