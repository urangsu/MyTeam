# MyTeam — Product Direction

**Updated: Round 243A-OBSERVE**

---

## 핵심 포지셔닝

MyTeam은 사용자가 보고 있는 파일/화면/폴더를 안전하게 읽어 업무 결과물로 바꾸는 생활·사무 AI 업무방이다.

---

## 핵심 사용자

- 사무직 직원 (회계/법무/기획/마케팅)
- 1인 창업자
- 콘텐츠 제작자
- 세무/회계/관리 업무 사용자
- 개발자/디자이너가 아니어도 반복 문서 업무가 많은 사람

---

## 핵심 루프

```
사용자가 파일을 드롭하거나 요청함
  ↓
MyTeam이 파일을 안전하게 읽음 (사용자 확인)
  ↓
적합한 skill/route 선택 (계정 검토, 계약서 체크, 보고서 정리 등)
  ↓
결과물(artifact) 생성
  ↓
다음 액션 제안
```

---

## 비교 포지셔닝

| 도구 | 강점 |
|---|---|
| Claude Code / Codex | 코드 전문, 파일 시스템 직접 조작 |
| MyTeam | 사무 문서, 회계/법무/업무 결과물, 자연어 요청, 팀원 캐릭터 경험 |

---

## 관찰 계층 (Round 243A 추가)

사용자 업무 자료를 안전하게 가져오는 채널:
- 파일 드롭 (즉시 감지)
- 다운로드 폴더 감지 (기본 OFF, 메타데이터만)
- 클립보드 읽기 (명시 요청만)
- Finder 선택 (명시 요청만, skeleton)
- 화면 스냅샷 (planned)

모든 관찰은 room-scoped. 자동 외부 업로드 금지.
