## MyTeam

### Positioning
**내 컴퓨터 안의 AI 업무 팀**

Killer Message: 회의록·체크리스트·보고서를 빠르게

### Primary Channels
- Naver Blog (SEO-first)
- Blind (1인 창업자/프리랜서 커뮤니티)
- Product Hunt (international launch)
- LinkedIn (B2B, later phase)

### Primary Audience
- 1인 창업자
- 프리랜서
- 소규모 팀 리더
- macOS 네이티브 앱 선호 사용자

### Key Messaging Pillars
1. **로컬 우선** — API key 없어도 시작
2. **빠른 초안** — 회의록/체크리스트/보고서
3. **내 데이터는 내 Mac에** — 자체 서버 미저장
4. **자동 실행 금지** — 위험한 작업은 승인 필요
5. **BYOK 확장** — 사용자가 선택한 API provider

### Success Metrics (6-month)
- macOS App Store: Top 100 Productivity
- Naver 자연 검색: 2,000+ traffic/month
- "맥북 AI 앱" 검색: Top 10 ranking
- Product Hunt: #1 Product of the Day (if possible)
- Active users: 500+

**Current**: `/Users/su/Desktop/MyTeam`

## IMMM (Instagram Micro-Moments)

### Positioning
**설치 없이 찍는 네컷 포토부스**

### Primary Channels
- Instagram Reels (main)
- TikTok (secondary)
- Naver Blog (evergreen guide)
- Everytime (대학생 커뮤니티)

### Primary Audience
- 대학생 (20-25)
- 커플
- 친구 그룹
- 이벤트 참가자

### Key Messaging Pillars
1. **간편함** — 앱 설치 없이 바로 사용
2. **감성** — 추억, 친구, 낭만
3. **공유 가능** — SNS 바로 올리기
4. **가성비** — 무료/저가

### Content Types
- Reels: 30-60초 사용자 후기
- TikTok: 자연스러운 '놀러갔다가 발견' 컨셉
- Blog: SEO/evergreen "포토부스 추천" 가이드
- Everytime: 친구 추천 글 (자연 발생식)

### Success Metrics (6-month)
- Instagram followers: 5,000+
- TikTok followers: 10,000+
- Naver Blog traffic: 1,000+/month
- Everytime 언급: 20+
- Web traffic from social: 100+/month

### Repository
**Separate**: IMMM product repo (not MyTeam)
**MyTeam Code Impact**: Zero

## When We Meet

### Positioning
**약속이 기다려지는 초대장**

### Primary Channels
- Naver Blog (culture/dating)
- Everytime (대학생 데이팅)
- Kakao Share (viral loop)
- Instagram Stories (teaser)

### Primary Audience
- 대학생 (20-25)
- Young professionals (25-35)
- Dating app users
- Event organizers

### Key Messaging Pillars
1. **낭만** — 약속을 특별하게
2. **감성** — 따뜻하고 꼼꼼한 초대
3. **유연함** — 누구나 쉽게 만들기
4. **공유** — 초대장이 SNS 될 때

### Content Types
- Blog: "초대장 만드는 방법" + "초대장 감상 모음"
- Everytime: "우리 만났던 날..." 회상 글
- Kakao Share: "너도 이렇게 초대받고 싶지?" 바이럴
- Instagram: 사용자 초대장 스크린샷 재공유

### Success Metrics (6-month)
- Naver Blog traffic: 500+/month
- Kakao Share conversions: 10+/month
- Web signups: 100+/month
- User-generated invitations: 50+
- Dating app cross-promotion: Partnerships 3+

### Repository
**Separate**: When We Meet product repo (not MyTeam)
**MyTeam Code Impact**: Zero

## Cross-Product Synergy

### MyTeam → IMMM
- MyTeam blog에서 "1인 창업자" 타깃 중 IMMM creator 추천
- IMMM creators가 MyTeam으로 일정 관리 (subtle)

### MyTeam → When We Meet
- MyTeam blog에서 "약속 관리" tip 언급 시 When We Meet 연결
- When We Meet users가 MyTeam으로 준비 checklist 만들기

### IMMM → When We Meet
- IMMM event에서 "우리 만난 날" When We Meet 초대장 활용

### When We Meet → MyTeam
- When We Meet users = 이벤트 오거나이저 = checklist/보고서 수요

## Important Guardrails

### MyTeam Code Repo
- IMMM/When We Meet 코드 변경 금지
- IMMM/When We Meet 마케팅용 feature 개발 금지
- 문서 전략만 작성 (코드 미작성)

### Brand Separation
- 3개 product logo/color 완전히 다름
- 3개 App Store page 완전히 독립적
- "내 컴퓨터 안의 AI" 위치만 MyTeam에만 사용

### No False Association
- MyTeam blog에서 IMMM/When We Meet 과도하게 언급 금지
- IMMM/When We Meet blog에서 MyTeam 강제 노출 금지
- 자연스러운 크로스-링크만 허용
