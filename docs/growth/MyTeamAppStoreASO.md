## Description

MyTeam은 macOS에서 문서 초안, 파일 정리, 오늘 할 일 브리핑을 도와주는 업무 보조 앱입니다.

### 특징
- **회의록 양식과 초안 생성** — 회의 내용을 붙여넣으면 정리된 초안이 나옵니다
- **체크리스트** — 앱 출시, 업무 인수인계, 회의 준비 같은 반복 업무를 정리합니다
- **보고서 초안** — 데이터와 배경을 바탕으로 검토용 보고서 초안을 생성합니다
- **파일 정리** — 텍스트, 마크다운, CSV 파일을 읽고 요약하거나 표로 정리합니다
- **오늘 할 일 브리핑** — 로컬 스케줄과 오늘의 주요 업무를 빠르게 확인합니다

### 사용 방식
로컬 문서 기능은 API key 없이 바로 시작할 수 있고, AI 기능은 사용자가 연결한 API provider를 통해 동작합니다.
MyTeam은 자신의 서버에 파일을 저장하지 않으며, 선택한 파일과 생성된 문서는 내 Mac의 Workspace에 저장됩니다.

### 안전성
메일을 자동 발송하거나, 일정을 자동 생성/수정하거나, 파일을 자동 삭제하지 않습니다.
위험한 작업은 항상 사용자의 명시적 승인이 필요합니다.

## Privacy & Safety Messaging

### How Data Flows
**로컬 기능** (API key 없음)
- 문서 템플릿, 로컬 파일 정리, 오늘 할 일 브리핑
- 모든 처리가 내 Mac에서만 진행됩니다

**AI 기능** (API key 필요)
- AI 기능을 사용할 때 선택한 텍스트나 파일이 사용자가 연결한 API provider로 전송될 수 있습니다
- 예: Claude API, OpenAI API 등
- MyTeam 자체 서버에는 전송되지 않습니다

**Workspace 저장**
- 생성된 문서와 artifact는 내 Mac의 Workspace 디렉토리에 저장됩니다
- Finder에서 직접 접근 가능합니다
