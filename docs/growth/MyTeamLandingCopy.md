## FAQ

### Q1: API key가 없어도 쓸 수 있나요?
네, 로컬 문서 템플릿과 파일 정리 기능은 API key 없이 바로 시작할 수 있습니다.

### Q2: 내 파일과 문서는 어디에 저장되나요?
내 Mac의 Workspace 폴더에 저장됩니다. MyTeam 자체 서버에는 저장되지 않습니다.

### Q3: AI 기능을 쓰려면 어떻게 하나요?
Claude, OpenAI 등 선택한 provider의 API key를 MyTeam 설정에서 연결하면 됩니다. 비용은 해당 provider의 요금 정책에 따릅니다.

### Q4: Gmail, Slack과 연동되나요?
현재는 텍스트 파일 중심입니다. 향후 확장될 예정입니다.

### Q5: Windows에서도 쓸 수 있나요?
현재는 macOS만 지원합니다.
